Culture discovery Platform

Aisira

The Challenge & Vision

Coastal Karnataka possesses highly rich traditional art and sports formats (like Yakshagana, Kambala, Nema/Kola). However, finding their locations, times, and dates historically depended on word-of-mouth or localized posters. Aisira acts as an enterprise-grade Progressive Web App that aggregates, maps, and broadcasts event details in real-time.

Architecture Highlights

- Offline Access: Developed a custom Service Worker utilizing cache-first strategical models to ensure maps and lists remain available even in regions with spotty cellular network coverage.
- Twilio Automation: Integrated Twilio's WhatsApp Business API, setting up automated messaging channels that dispatch schedules, route updates, and location updates.
- Interactive Maps: Engineered dynamic coordinate-placements via Leaflet.js with an automated administrative coordinate-extraction dashboard.
- Assets: Utilized Cloudinary for fast media manipulation and high-speed multi-poster gallery displays.

The Scope

The Courtyard is an integrated club booking and commercial management platform designed to automate operations. It handles membership billing, real-time sports court slot bookings, split bills at physical point-of-sale systems, and tracks operational inventory in real-time.

Technical Implementation

- Concurrency Protection: Structured slot reservations with robust transaction locking in MongoDB to prevent double-bookings of sports facilities.
- Peak Pricing Algorithms: Built a dynamic pricing module adjusting bookings costs based on hour categories and weekend demand models.
- Secured Admin Workspace: Formed charts and master ledger accounting systems using Chart.js, providing managers precise visual analytics over cash flows, booking densities, and high-frequency time brackets.

Capstone Vision

Built as the final capstone project during the 16-week industrial internship at SuprMentr Technologies. The program demanded the creation of a high-concurrency, secure multi-role exam administration site.

Key Security Integrations

- Anti-Cheat Monitoring: Developed a strict tab-activity observer system. If a student leaves the testing browser tab, the application records a tab-blur infraction, flashing alert logs to administrators and locked accounts if threshold limits are exceeded.
- Role-Based Security (RBAC): Formulated clean database routing filters categorizing accounts into: Student (test taking), Quizmaster (test creation, validation), and Administrator (global monitoring, analytical graphs).
- Bcrypt & JWT Encryption: Structured high-security sign-ins with salted token exchanges and encrypted local user configurations.
