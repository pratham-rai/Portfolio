Secure Quizmaster

Industry-grade multi-role testing software built as the capstone project during the 16-week industrial internship at SuprMentr Technologies.

React Node.js Express MongoDB Atlas JWT Bcrypt

The Challenge & Vision

Online testing platforms often sacrifice security for usability, leaving exam integrity vulnerable to tab-switching, browser manipulation, and shared credentials. Secure Quizmaster was built to solve this — delivering an enterprise-grade assessment experience that enforces test integrity through real-time activity monitoring, while remaining intuitive for students, quiz creators, and administrators alike.

Developed as the capstone project during a rigorous 16-week MERN Stack internship at SuprMentr Technologies, this PWA demonstrates production-level engineering across authentication, authorization, real-time data, and deployment pipelines.

Architecture & Key Features

  • Anti-Cheat Tab-Activity Observer
    Monitors browser visibility state and tab-switching events in real-time. Logs every focus/blur transition, flags suspicious patterns, and can auto-submit tests after configurable violation thresholds.
  • Role-Based Access Control (RBAC)
    Three distinct authorization tiers — Student, Quizmaster, and Administrator — each with precisely scoped API endpoints, UI views, and data access policies enforced via middleware guards.
  • Bcrypt & JWT Encryption
    Passwords hashed with Bcrypt (configurable salt rounds). Stateless session management via signed JWTs with automatic expiry, refresh token rotation, and secure httpOnly cookie storage.
  • Timed Test Sequences
    Server-synchronized countdown timers that cannot be manipulated client-side. Tests auto-submit on expiry regardless of client state, with grace-period handling for network latency.
  • Real-Time Leaderboards
    Live score rankings that update as students complete tests. Aggregated analytics dashboards for quizmasters showing score distributions, average times, and question-level performance heatmaps.
  • Deployed on Vercel & Render
    React frontend hosted on Vercel with edge CDN delivery. Express API deployed on Render with auto-scaling and health monitoring. MongoDB Atlas for cloud-native database hosting.
