Cultural Event Discovery Platform

Aisira

Enterprise Progressive Web App facilitating cultural event discovery (Yakshagana, Kambala, Nema/Kola) across Coastal Karnataka.

Vite Node.js Express MongoDB Atlas Leaflet.js Twilio Cloudinary

The Challenge & Vision

Coastal Karnataka is home to extraordinarily rich traditional art forms and sporting events — Yakshagana (ancient mythological dance-drama), Kambala (buffalo racing), and Nema/Kola (spirit worship rituals). Yet for decades, discovering when and where these events take place relied entirely on word-of-mouth, hand-pasted posters, and local networks.

Aisira was envisioned as an enterprise-grade Progressive Web App that aggregates, geo-maps, and broadcasts cultural event details in real-time — democratizing access for locals and tourists alike, while empowering event organizers with a self-service admin dashboard.

Architecture & Key Features

  • Offline Service Worker (Cache-First Strategy)
    Custom Service Worker ensures maps, event lists, and core UI remain fully available in areas with spotty or non-existent cellular coverage — critical for rural Coastal Karnataka.
  • Twilio WhatsApp Business API Integration
    Automated messaging pipelines dispatch event schedules, location pin updates, and route directions directly to subscribed users via WhatsApp — zero app-install friction.
  • Interactive Leaflet.js Maps
    Dynamic coordinate placement with custom event-category markers and an admin dashboard featuring click-to-extract coordinate tooling for rapid event geo-tagging.
  • Cloudinary Media Management
    Optimized asset delivery with on-the-fly image transformations, ensuring fast multi-poster gallery loading and responsive image sizing across all devices.
